1.プロジェクト概要
横浜市に関するクイズを楽しめるiOSアプリです。SwiftUIとMVC設計を採用し、Appstoreに公開済みです。
2.開発環境
Swift/SwiftUI/Xcode/MVC/GoogleAdmob
3.主要機能
- 横浜に関するクイズ出題（4択形式）
- 正答/不正解の判定とスコア計算
- スコア表示画面
- Google AdMobによる広告表示
